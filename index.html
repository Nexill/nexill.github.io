🛒 Retail Store POS by Nexill

A private, commercial Windows desktop application built for professional retail checkout operations.

📥 Download Latest Installer (v1.0.0)

Retail Store POS was designed for one primary reason: your store should keep selling even when internet service is unstable. The application runs entirely locally, ensuring that cashiers can continue processing sales, calculating tax, and printing receipts without ever relying on cloud uptime.

✅ What You Get

  • Fast checkout flow optimized for day-to-day cashier use.
  • Fully offline operation secured entirely on your local register.
  • Simple tax setup, including customizable tax-exempt item support.
  • PDF receipt generation directly integrated with native Windows print support.
  • Manager-only controls to lock down sensitive store configuration settings.

💻 System Requirements

  • OS: Windows 10 or Windows 11 (64-bit)
  • Hardware: Standard PC, laptop, or Windows-based POS terminal
  • Optional: Physical receipt printer

📥 How to Install

  1. Click the large green download button at the top of this page.
  2. Download the RetailStorePos_Setup.exe file.
  3. Run the installer and follow the setup wizard.
  4. Launch Retail Store POS from your Start Menu or Desktop.
